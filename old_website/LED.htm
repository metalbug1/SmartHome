General Purpose I/O Ports (PORTS)

The XMC4500 has many digital port pins which can provide a generic and flexible software and hardware interface for all standard digital I/Os. Each port slice has the same software interfaces for the operation as General Purpose I/O and it further provides the connectivity to the on-chip periphery and the control for the pad characteristics.

Demo Overview

This demo make use of the Common Gate Way Interface (CGI) using the GET method to request the toggling of the respective LEDs.
